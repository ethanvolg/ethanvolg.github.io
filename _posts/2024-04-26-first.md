# 1. SQL 이란?

Structured Query Language의 약자입니다.

Structured : 관계형 데이터(table형태의 데이터)  
Query : 질의, 요청(ex. 프로그램 언어를 통해 web에서 클라이언트가 요청 )   
Language : 프로그램 언어

정리해보면 관계형 데이터를 목적에 맞게 요청할 수 있는 프로그램 언어라고 볼 수 있습니다.
